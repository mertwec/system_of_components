Create "report file" from P-Cad:

Before creating: bring components to the same form in "____.pcb" file:
Correct pattern, value etc.

open pcb-file in P-CAD 2006 PCB

  1. menu "File"
  2. "Report...":
    • Reports to generate: Bill of Materials(bom)
    • Report Destination: File
    • Style Format: Separated List
    • List Separator: ";"
  3. button "Customize...":
    • "Format": File Extension: 'csv'
    • "Selection": 'Count', 'RefDes', 'PatternName' 'Value'
    • "Ok"
  4. button "Generate" and "Close"
  5. Open and check created report file
